## ② 内容（どんな作品か）

### 衛生管理アプリ
- 衛生管理項目のフォームをテンプレから作成できる。例えば管理者/従業員、出勤時/退勤時に応じて必要な衛生管理項目のフォームを作成できる。
- 項目ごとに、チェックが必要か、温度の入力が必要か、画像が必要かは異なる。
- よって、衛生管理項目が増えても管理項目のフォームを作成できるので、柔軟に対応できる汎用性を持たせた。
